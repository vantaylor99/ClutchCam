The transcript prefilter now recognizes the missed live gaming callouts from the
medium Faster-Whisper evaluation without lowering the global transcript length
guard.

Implemented behavior:

- Added default longer gaming callout phrases: "that was nasty", "behind you",
  "good work", "triple", and "legend".
- Added a separate exact-match short callout list for "help", so a lone "help"
  can trigger while short filler such as "ok" remains rejected.
- Gaming callout matches now produce a specific local reason string:
  `Matched gaming callout phrase: <phrase>.`
- Existing duplicate suppression continues to suppress repeated matching
  phrases across nearby transcript events and across streams, including the new
  "behind you" and exact "help" cases.
- The runtime transcript event path now has coverage proving an accepted event
  containing "that was nasty" reaches `ai_director.decide(...)` with a
  `candidate_signal`.
